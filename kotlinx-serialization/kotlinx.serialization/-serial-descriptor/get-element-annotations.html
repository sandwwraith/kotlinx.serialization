getElementAnnotations

(common, js, jvm, native) abstract fun getElementAnnotations(index: Int): List<Annotation>

Returns serial annotations of the child element at the given index. This method differs from getElementDescriptor(index).annotations as it reports only declaration-specific annotations:

@Serializable
@SerialName("_nested")
class Nested(...)
@Serializable
class Outer(@SerialId(1) val nested: Nested)

outerDescriptor.getElementAnnotations(0) // Returns [@SerialId(1)]
outerDescriptor.getElementDescriptor(0).annotations // Returns [@SerialName("_nested")]

This method is guaranteed to be consistent with elementsCount.

Exceptions

IndexOutOfBoundsException - for an illegal index values.

IllegalStateException - if the current descriptor does not support children elements (e.g. is a primitive).
