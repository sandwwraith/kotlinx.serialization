getElementName

(common, js, jvm, native) abstract fun getElementName(index: Int): String

Returns a positional name of the child at the given index. Positional name usually represents a corresponding property name in the class, associated with the current descriptor.

This method may be inconsistent with elementsCount for collection and map types. Refer to its documentation for more details.

Exceptions

IndexOutOfBoundsException - for an illegal index values.

IllegalStateException - if the current descriptor does not support children elements (e.g. is a primitive)
