UnknownFieldException

(common, js, jvm, native) class UnknownFieldException : SerializationException

Thrown when KSerializer received unknown property index from CompositeDecoder.decodeElementIndex.

Usually this exception means that data schema has changed in backwards-incompatible way. Index may be negative, in that case it should be equal to CompositeDecoder.UNKNOWN_NAME.
