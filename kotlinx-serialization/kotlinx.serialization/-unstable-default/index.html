UnstableDefault

(common, js, jvm, native) annotation class UnstableDefault

This annotation marks declarations with default parameters that are subject to semantic change without a migration path.

For example, JsonConfiguration.Default marked as unstable, thus indicating that it can change its default values in the next releases (e.g. disable strict-mode by default), leading to a semantic (not source code level) change.
