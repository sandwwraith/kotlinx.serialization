(common, js, jvm, native) fun <reified T> serializer(): KSerializer<T>(common, js, native)

Creates a serializer for the provided reified type T with support of user-defined generic classes. This method is a reified version of serializer(KType)

Example of usage:

val map = mapOf(1 to listOf(listOf("1")))
json.stringify(serializer(), map)

This is a computation-heavy call, so it is recommended to cache its result. typeOf API currently does not work with user-defined generic classes on Kotlin/JS.

(common, js, jvm, native) fun serializer(type: KType): KSerializer<Any?>(common, js, native)

Creates a serializer for the given type with support of user-defined generic classes. type argument can be obtained with experimental typeOf method.

Example of usage:

val map = mapOf(1 to listOf(listOf("1")))
val serializer = serializer(typeOf<Map<Int, List<List<String>>>>())
json.stringify(serializer, map)

This is a computation-heavy call, so it is recommended to cache its result. typeOf API currently does not work with user-defined generic classes on Kotlin/JS.
