Retrieves a KSerializer for the given KClass. The class must be annotated as @Serializable or be one of the built-in types.

This method has all the restrictions implied by ImplicitReflectionSerializer, i.e., it will fail on an attempt to retrieve a serializer for generic class. For classes with type parameters serializer(KType) should be used instead.

Exceptions

SerializationException - if serializer can't be found.
