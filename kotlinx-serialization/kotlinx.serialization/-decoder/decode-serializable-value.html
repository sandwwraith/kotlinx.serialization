decodeSerializableValue

(common, js, jvm, native) open fun <T> decodeSerializableValue(deserializer: DeserializationStrategy<T>): T

Decodes the value of type T by delegating the decoding process to the given deserializer. For example, decodeInt call us equivalent to delegating integer decoding to IntSerializer: decodeSerializableValue(IntSerializer)
