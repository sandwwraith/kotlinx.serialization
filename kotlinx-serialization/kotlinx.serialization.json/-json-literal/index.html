JsonLiteral

(common, js, jvm, native) class JsonLiteral : JsonPrimitive

Class representing JSON literals: numbers, booleans and string. Strings are always quoted.

isString indicates whether literal was explicitly constructed as a String and whether it should be serialized as one. E.g. JsonLiteral("42", false) and JsonLiteral("42", true) will be serialized as 42 and "42" respectively.
